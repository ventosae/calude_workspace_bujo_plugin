Run the daily ritual using the `daily` skill.

Read yesterday's daily log from `bujo/daily/` and today's if it exists.
Check `bujo/capture/inbox.md` for any new captures.

Create today's log at `bujo/daily/YYYY-MM-DD.md` if it doesn't exist.

Then: review incomplete tasks from yesterday, rapid log any new captures, set 3 intentions for today, record one gratitude note.

Close with: "Three things today: [1], [2], [3]."
